  POST /api/v1/agents/{agent_id}/run — 에이전트 실행

  에이전트 정의를 DB에서 로드하고 LangGraph 워크플로우를 동적 컴파일하여 응답을 생성한다.

  Authentication

  - Authorization: Bearer {access_token} (필수)

  Path Parameters

  ┌──────────┬────────┬──────────┬────────────────────┐
  │   Name   │  Type  │ Required │    Description     │
  ├──────────┼────────┼──────────┼────────────────────┤
  │ agent_id │ string │ Yes      │ 실행할 에이전트 ID │
  └──────────┴────────┴──────────┴────────────────────┘

  Request Body

  {
    "query": "string",       // 필수, 1~2000자
    "user_id": "string",     // 필수
    "session_id": "string"   // 선택 (null이면 새 세션 생성)
  }

  ┌────────────┬───────────────┬──────────┬─────────────────┬──────────────────────────────────────────┐
  │   Field    │     Type      │ Required │   Constraints   │               Description                │
  ├────────────┼───────────────┼──────────┼─────────────────┼──────────────────────────────────────────┤
  │ query      │ string        │ Yes      │ min 1, max 2000 │ 사용자 질문/명령                         │
  ├────────────┼───────────────┼──────────┼─────────────────┼──────────────────────────────────────────┤
  │ user_id    │ string        │ Yes      │                 │ 사용자 ID                                │
  ├────────────┼───────────────┼──────────┼─────────────────┼──────────────────────────────────────────┤
  │ session_id │ string | null │ No       │                 │ 기존 대화 세션 ID. null이면 새 세션 시작 │
  └────────────┴───────────────┴──────────┴─────────────────┴──────────────────────────────────────────┘

  Response 200 OK

  {
    "agent_id": "abc-123",
    "query": "원래 보낸 질문",
    "answer": "에이전트 응답 텍스트",
    "tools_used": ["internal_document_search", "web_search"],
    "request_id": "uuid-xxx",
    "session_id": "session-uuid"
  }

  ┌────────────┬──────────┬─────────────────────────────────┐
  │   Field    │   Type   │           Description           │
  ├────────────┼──────────┼─────────────────────────────────┤
  │ agent_id   │ string   │ 실행된 에이전트 ID              │
  ├────────────┼──────────┼─────────────────────────────────┤
  │ query      │ string   │ 원본 질문 (echo)                │
  ├────────────┼──────────┼─────────────────────────────────┤
  │ answer     │ string   │ 에이전트 최종 응답              │
  ├────────────┼──────────┼─────────────────────────────────┤
  │ tools_used │ string[] │ 실행 중 사용된 도구 ID 목록     │
  ├────────────┼──────────┼─────────────────────────────────┤
  │ request_id │ string   │ 요청 추적 UUID                  │
  ├────────────┼──────────┼─────────────────────────────────┤
  │ session_id │ string   │ 대화 세션 ID (멀티턴 시 재사용) │
  └────────────┴──────────┴─────────────────────────────────┘

  Error Responses

  ┌──────────────────────────┬────────────────────────────────────────────────┬───────────────────────────────┐
  │          Status          │                   Condition                    │             Body              │
  ├──────────────────────────┼────────────────────────────────────────────────┼───────────────────────────────┤
  │ 403 Forbidden            │ 실행 권한 없음 (비공개 에이전트, 부서 제한 등) │ {"detail": "실행 권한 없음"}  │
  ├──────────────────────────┼────────────────────────────────────────────────┼───────────────────────────────┤
  │ 404 Not Found            │ agent_id에 해당하는 에이전트 없음              │ {"detail": "..."}             │
  ├──────────────────────────┼────────────────────────────────────────────────┼───────────────────────────────┤
  │ 422 Unprocessable Entity │ 유효성 검증 실패 (query 빈 값, 길이 초과 등)   │ FastAPI 기본 validation error │
  └──────────────────────────┴────────────────────────────────────────────────┴───────────────────────────────┘

  멀티턴 대화 흐름

  1회차: session_id = null → 응답의 session_id 수신
  2회차~: session_id = 받은 값 전달 → 이전 대화 컨텍스트 유지

  cURL 예시

  curl -X POST http://localhost:8000/api/v1/agents/abc-123/run \
    -H "Authorization: Bearer {token}" \
    -H "Content-Type: application/json" \
    -d '{
      "query": "최근 대출 금리 동향 알려줘",
      "user_id": "user-001",
      "session_id": null
    }'
