## 1. 개요 (Overview)

현재 TopNav(상단 메뉴) + Sidebar(채팅 세션 목록) + Main(채팅 영역)의 3단 구조를
Deep Agent Builder 스타일의 **왼쪽 네비게이션 사이드바 + 토글 가능한 채팅 히스토리 패널 + 메인 채팅 영역** 구조로 전환한다.

## 2. 현재 상태 분석 (As-Is)

### 현재 레이아웃 구조

```
App.tsx
├── TopNav (상단 가로 메뉴바 — 데이터/에이전트/도구/관리/기타 드롭다운)
└── AuthenticatedLayout
    ├── TopNav
    └── <Outlet> (각 페이지)
         └── ChatPage
              ├── Sidebar (w-64, 검은 배경, 채팅 세션 목록 + 하단 네비)
              └── Main (채팅 영역)
```

### 현재 문제점

1. **TopNav와 Sidebar 중복 네비게이션**: TopNav에 드롭다운 메뉴, Sidebar 하단에도 네비 아이템이 있어 중복
2. **에이전트 선택 불가**: 현재 "SUPER AI Agent" (일반 채팅)만 존재, 에이전트를 선택해서 대화하는 구조 아님
3. **채팅 히스토리가 항상 노출**: 사이드바에 채팅 세션 목록이 고정되어 있어 화면 활용도 낮음
4. **페이지별 분리**: ChatPage, AgentBuilderPage 등이 완전히 다른 페이지로 분리되어 있어 에이전트 ↔ 채팅 간 전환 불편

### 영향받는 파일

| 파일 | 변경 유형 | 설명 |
|------|----------|------|
| `src/App.tsx` | **Major** | 레이아웃 구조 전면 변경 (TopNav 제거 또는 조건부) |
| `src/components/layout/Sidebar.tsx` | **Replace** | 새로운 AppSidebar로 교체 |
| `src/components/layout/TopNav.tsx` | **Remove/Reduce** | 사이드바로 네비 이관, TopNav 제거 또는 축소 |
| `src/pages/ChatPage/index.tsx` | **Major** | 3단 레이아웃 통합, 에이전트 선택 연동 |
| `src/types/agent.ts` | **Extend** | 에이전트 목록 타입 추가 |
| `src/constants/api.ts` | **Extend** | 에이전트 목록 API 엔드포인트 추가 |

## 3. 목표 상태 (To-Be)

### 3-1. 레이아웃 구조

```
App.tsx
├── LoginPage / RegisterPage (공개 라우트, 기존 유지)
└── ProtectedRoute
    └── AgentChatLayout (새 레이아웃 — TopNav 제거)
        ├── AppSidebar (왼쪽 사이드바, 항상 노출)
        │   ├── 로고 영역
        │   ├── 네비게이션 (SUPER AI, 에이전트 템플릿, 유틸리티, 작업, 평가)
        │   ├── 에이전트 섹션 (서버 조회 → Mock)
        │   │   ├── 액션바 (숨기기/필터/복사/새로고침)
        │   │   └── 카테고리별 에이전트 목록
        │   └── 하단 메뉴 (즐겨찾기, 역할설정, 리소스, 환경설정)
        ├── ChatHistoryPanel (토글 가능한 중간 패널)
        │   ├── 새 채팅 버튼
        │   ├── 채팅 검색
        │   └── 세션 목록
        └── MainChatArea (메인 채팅 영역)
            ├── 에이전트 헤더 (선택된 에이전트 이름 + 설명)
            ├── 메시지 목록
            └── 입력창 (모델 선택 드롭다운 포함)
```

### 3-2. 에이전트 모델

```typescript
// 사이드바에 표시할 에이전트 목록용 타입
interface AgentSummary {
  id: string;
  name: string;
  description: string;
  category: string;       // "미분류" | 커스텀 카테고리
  isDefault?: boolean;     // SUPER AI Agent 여부
}
```

- **SUPER AI Agent**: 현재의 일반 채팅(`/api/v1/conversation/chat`)을 SUPER AI Agent로 취급
- **커스텀 에이전트**: AgentBuilderPage에서 만든 에이전트들 → 목록에 표시 (Mock)
- 에이전트 선택 시 해당 에이전트와 대화하는 형태 (1차에서는 SUPER AI Agent만 실제 동작)

### 3-3. 채팅 히스토리 패널

- **토글 버튼**: 사이드바 상단 or 메인 영역 좌상단에 배치
- **열린 상태**: 너비 288px, 세션 목록 + 검색 표시
- **닫힌 상태**: 패널 숨김, 메인 영역이 전체 너비 활용
- 기존 Sidebar의 세션 목록 로직을 이 패널로 이관

### 3-4. 비에이전트 페이지 라우팅

에이전트/채팅 외 페이지(컬렉션, 워크플로우, 도구 관리 등)는 사이드바 네비에서 접근하되,
메인 영역이 해당 페이지로 전환되는 방식을 유지한다.

| 네비 항목 | 라우트 | 메인 영역 컨텐츠 |
|----------|--------|-----------------|
| SUPER AI 에이전트 | `/chatpage` | 채팅 UI (기본) |
| 에이전트 템플릿 | `/agent-builder` | 에이전트 빌더 |
| 유틸리티 | `/tool-connection` | 도구 연결 |
| 작업 | `/workflow-designer` | 워크플로우 |
| 평가 | `/eval-dataset` | 평가 데이터셋 |
| 컬렉션 관리 | `/collections` | 컬렉션 페이지 |
