# Skills Registry

> 원본: CLAUDE.md §14

AI가 사용할 수 있는 검증 스킬 목록입니다.

| 스킬 | 설명 |
|------|------|
| `chunk` | 문서 청킹 모듈 TDD 개발 (청킹, 분할, 토큰 전략 구현 시 사용) |
| `tdd` | TDD 방식으로 새 기능 개발. 테스트 먼저 작성 → 실패 확인 → 구현 → 통과 확인 사이클 강제 |
| `langgraph` | LangChain/LangGraph 기반 AI 에이전트 및 그래프 워크플로우 개발 |
| `langchain-middleware` | LangChain 미들웨어 추가/설정 |
| `verify-architecture` | DDD 레이어 의존성 규칙 준수 검증 |
| `verify-logging` | LOG-001 로깅 규칙 준수 검증 |
| `verify-tdd` | 프로덕션 모듈 테스트 파일 존재 여부 검증 |
| `verify-implementation` | 모든 verify 스킬을 순차 실행하여 통합 검증 보고서 생성 |
| `manage-skills` | 세션 변경사항 분석으로 검증 스킬 누락 탐지, 스킬 생성/업데이트 |
